# Coffee Shop Sales Analysis

## Overview

This project analyzes coffee shop sales using Python and SQL.

## Tools

- Python
- Pandas
- SQL
- Matplotlib
- Excel
- GitHub

## Features

- Data Cleaning
- Data Analysis
- SQL Queries
- Charts
- Business Insights

## Business Insights

- Identified top-selling coffee products.
- Calculated total revenue.
- Compared payment methods.
- Analyzed daily sales trends.

## How to Run

```bash
python src/data_cleaning.py
python src/analysis.py
```
